Detailed Description

The Zombie class represents a zombie ghost in the game.

This class inherits from the Ghost class and provides functionalities and properties for the zombie ghost.

Checks for prey near the zombie ghost on the game board.

This function checks if the Zombie sees Pacman in his line of sight. If yes, he goes in that direction until he either catches Pacman or hits the wall.
